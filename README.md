# **KeysocAlbum**

Demo App for Keysoc

## Tech Stack

- UIKit + RxSwift + MVVM
- package manager: **CocoaPods**
- minimun deployment target: **iOS 14.5**

## Dependency

- Alamofire : Network Library
- Kingfisher : Image Caching
- RxSwift : Reactive Programming Library
- SnapKit : Autolayout Helper
